## 🎮 Games Overview

### 1️⃣ Rocket Flight 🚀

```
OBJECTIVE: Reach maximum altitude without crashing

CONTROLS:
  [SPACEBAR] = Fire engines (costs fuel)
  [RELEASE]  = Fall under gravity

MECHANICS:
  • Fuel system (depletes when firing, recharges when idle)
  • Gravity pulls you down constantly
  • Obstacles move left across screen
  • Score = Maximum altitude reached

PHYSICS:
  Gravity ↓↓↓ 0.45
  Thrust ↑↑↑ 0.6
  = Challenge level: MEDIUM-HARD

VISUAL:
  • Animated engine flames
  • Real-time velocity display
  • Fuel bar (green → yellow → red)
  • Grid background
  • Gradient sky
```

### 2️⃣ Snake Game 🐍

```
OBJECTIVE: Eat food and grow longer without hitting walls

CONTROLS:
  W = Move Up    ↑
  A = Move Left  ←
  S = Move Down  ↓
  D = Move Right →

MECHANICS:
  • Snake grows each time it eats
  • Score increases with length
  • Touching walls = Game Over
  • Touching self = Game Over
  • Food spawns randomly

VISUAL:
  • Grid-based movement
  • Classic arcade style
  • Score display
  • High score tracking

DIFFICULTY: EASY → MEDIUM (gets harder as you grow)
```

### 3️⃣ Memory Match 🧠

```
OBJECTIVE: Find all matching pairs before running out of moves

MECHANICS:
  • Click cards to flip and reveal
  • Find matching pairs
  • Fewer moves = higher score
  • All pairs matched = Victory

VISUAL:
  • Smooth card flip animation
  • Pairs stay face-up when matched
  • Color feedback (correct/incorrect)
  • Move counter

DIFFICULTY: EASY
BEST FOR: Quick games, memory training
```

### 4️⃣ Quiz Master ❓

```
OBJECTIVE: Answer 40 questions correctly

CATEGORIES (5 questions each):
  📍 Geography     - World capitals, landmarks
  🔬 Science       - Space, physics, chemistry
  📖 Literature    - Books, authors, paintings
  📜 History       - Events, dates, figures
  🧬 Biology       - Human body, ecosystems
  💻 Technology    - Computers, internet, phones
  ⚽ Sports        - Games, athletes, records
  🎬 Entertainment - Movies, actors, box office

FEATURES:
  ✅ Questions shuffle every game (different order)
  ✅ Multiple choice (4 options each)
  ✅ Instant feedback (correct/incorrect)
  ✅ Final accuracy score (out of 100%)
  ✅ Score tracking for leaderboard

SCORING:
  • 10 points per correct answer
  • Maximum: 400 points
  • Shows: Score + Correct Count + Accuracy %

DIFFICULTY: MEDIUM (general knowledge)
```
